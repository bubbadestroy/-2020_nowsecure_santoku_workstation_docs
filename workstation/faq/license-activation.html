License Activation

License Activation

Online Activation:

  1. Double-click on the “NowSecure Activation Wizard” icon on the Desktop.

  2. Click “Forward” and select the “Online” activation method. From there, you will need to create a license container that will be used for activation. Click “Create License Container” and it should automatically populate in the drop down menu. Click “Forward.”


3. You will see an entry to enter a ticket number. You can copy/paste directly from the Welcome Email, or you can enter the Ticket Number manually (without dashes). Once complete, click “Continue” to have NowSecure Workstation initiate the activation process. Once complete, you will need to reboot the VM/System to complete the activation process.


Offline Activation:

  1. Double-click on the “NowSecure Activation” icon on the Desktop.

  2. Click “Forward” and select the “Offline” activation method. From there, you will need to create a license container that will be used for activation. Click “Create License Container” and it should automatically populate in the drop down menu. Click “Forward.”


3. You will see two options. Step 1 is to create a license request file, which will be needed by NowSecure Support in order to activate this instance of NowSecure Workstation. Select the first option, and click “Forward” to continue.


4. Once complete, a file with an RaC extension will be added to the desktop. This file will need to be emailed to support@nowsecure.com for activation. In return, you will receive a .RaU file which will contain the license update. Once you have the update file, you can continue the activation wizard.

Once the license has been activated, you can move on to Running NowSecure Workstation for the First Time.
