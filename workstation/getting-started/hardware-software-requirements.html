Hardware / Software Requirements

NowSecure Workstation System Requirements:

NowSecure Workstation is designed to run on Ubuntu 16.04, and is distributed as a Virtual Machine (VM). The VM has been pre-loaded with all the needed FOSS tools to streamline mobile application security testing, while minimizing the time spent configuring different tools.

Purchases of NowSecure Workstation have the option of including a Macbook that comes pre-provisioned and ready to go out of the box. In these instances, the VM is loaded and configured following the listed specifications below.

For setups that do not include the pre-provisioned Macbook the following specifications are recommended:

OS

  • VM Instance running using VMware Fusion (8.5+) with a minimum of 4GB of assigned virtual memory.
  • Based on Ubuntu 16.04 (64-bit)
  • Mac OS X 10.12 or later (host OS)

System

  • Apple Macbook Pro with equivalent or better specifications:
  • 2.4GHz+ Processor
  • 8GB memory
  • 256GB HDD Space
  • VMware Fusion (8.5 or later)

USB

  • (Min) Two (2) USB 2.0 or 3.0 ports

Internet Adapter

  • USB Wi-Fi adapter provided
  • Ethernet or WiFi connection (USB WiFi adapter included with purchase)

Installing VMware Fusion on Mac OSX:

If you already have VMWare Fusion installed, please skip ahead to Step 1: Obtaining NowSecure Workstation

If you have a Macbook that you plan to use with NowSecure Workstation, the next step is to install VMware Fusion. You can either obtain a license key directly from VMware, or sign-up for a 30-day trial. In either case, you will need to obtain the software:

http://www.vmware.com/try-vmware

Once on the page, scroll down to the “Personal Desktop” heading and select “VMware Fusion.” The download will begin momentarily.

When the download completes, double-click the VMwareFusion.dmg file to being the installation. From there, follow the on-screen instructions to complete the install. When complete, you are ready to move on to Step 1.
